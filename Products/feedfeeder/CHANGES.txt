With this system feed items data will be saved in a objectInfo field. It memoize the entry data, a dict.
Changing the remote RSS template, you will memoize different data without having to change the feed item schema.

If you want to index some value, you will be able to register a catalog indexer like that:

<adapter factory='.indexers.start' name='start' />

@indexer(IFeedItem)
def start(obj):
    object_info = obj.getObjectInfo()
    if object_info.get('portal_type') == 'Event':
        if 'startdate' in object_info:

            value = object_info['startdate']
            if isinstance(value, DateTime):
                return value
            else:
                return handle_date(value)
    return None
